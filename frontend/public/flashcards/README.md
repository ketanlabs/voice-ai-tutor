# Flashcard images

Picture assets for the pronunciation exercise. Referenced from
`backend/curriculum/exercise.yaml` via `/flashcards/<name>.jpg`.

The starter images were sourced via [Openverse](https://openverse.org) under the
**CC0** (public domain) license. Replace or add your own freely — any web image
format works (`.jpg/.png/.webp/.svg`). If an image is missing, the UI falls back to
the item's `emoji` + text, so the exercise never breaks.
